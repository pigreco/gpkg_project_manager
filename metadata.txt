about[en]=GeoPackage Project Manager is a plugin that allows complete management of QGIS projects stored within GeoPackage files.

    Main features:
    • Save and load QGIS projects in GeoPackage
    • Complete management: rename, duplicate, overwrite, delete projects
    • Export projects to QGS or QGZ format
    • Smart GeoPackage cloning with automatic path updates
    • Modern and intuitive interface with Qt5/Qt6 support
    • Integrated language selector in the interface (Italian/English)
    • Automatic detection of GeoPackages used in the current project

    The plugin is particularly useful for:
    • Organizing multiple projects within a single GeoPackage
    • Creating backups and project versions
    • Sharing complete projects with integrated data
    • Migrating projects between different environments while maintaining correct references
